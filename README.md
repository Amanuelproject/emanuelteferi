
# Vite + React static website prepared for universal static hosting.


## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Local Development

1. Clone the repository using your Git URL.
2. Navigate to the project directory.
3. Install dependencies: `npm install`
4. Start the development server: `npm run dev`

## Deployment

This project is ready for universal static hosting and GitHub Pages. No Lovable dependencies or branding remain.
